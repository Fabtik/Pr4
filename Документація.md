
## Графічний календар з замітками

### Огляд

Графічний календар з замітками - це програма, яка надає можливість користувачам створювати замітки та призначати їх дату на календарі. Програма використовує графічний інтерфейс, побудований з використанням бібліотеки `tkinter`, та зберігає замітки у вказаному файлі бази даних.

### Вимоги

Перед використанням програми переконайтеся, що у вас встановлені такі бібліотеки:

- `tkinter`: для створення графічного інтерфейсу.
- `tkcalendar`: для відображення календаря в програмі.
- `json`: для роботи з файлами JSON.

### Встановлення

Для встановлення необхідних бібліотек використовуйте команди:

```
pip install tk 
pip install tkcalendar
```

### Використання

1. Запустіть програму, виконавши файл `main.py`.
2. Відкриється графічний інтерфейс з календарем.
3. Виберіть дату, для якої ви хочете додати замітку.
4. Введіть текст замітки в поле для введення.
5. Натисніть кнопку "Додати замітку", щоб зберегти замітку для обраної дати.
6. Щоб переглянути всі замітки для вибраної дати, натисніть кнопку "Переглянути замітки".
7. Замітки відображатимуться у вікні сповіщень.

### Формат файлу бази даних

Програма зберігає замітки у форматі JSON у файлі бази даних (за замовчуванням - "notes.json"). Кожна дата має відповідні замітки у вигляді списку.

Приклад файлу бази даних:

Json
```json
{ 
	"01/01/2023": ["Спільне святкування", "Прийняти новорічні резолюції"], 
	"02/01/2023": ["Початок тренувань", "Подати податкову декларацію"] 
}
```


### Помітки

- Якщо файл бази даних не існує, програма створить його автоматично під час першого запуску.
- Поле для введення замітки не може бути порожнім, інакше вас сповістять про помилку.
